# ğŸ“– [React Weather App] <a name="about-project"></a>

**[React Weather App]** provides a user with current weather conditions, hourly and daily forecasts, and location-based weather information. The weather data will be fetched using two API. The search button allows users to enter the specific city and returns the weather results for that particular city. The React select async paginate is used to search for the specific city.
